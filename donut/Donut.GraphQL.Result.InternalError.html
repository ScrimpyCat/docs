    Donut vdev Donut.GraphQL.Result.InternalError View Source

    An internal error representation for result errors.

    • :message - Contains the presentable error message.
    • :exception - Contains the presentable exception module.
    • :stacktrace - Contains the presentable stacktrace from where the exception was raised/trown.
    • :error_message - The full error message that would have been displayed normally.
