    authenticate(arg)

    Implement the behaviour for authenticating an identity using the given credential.

    If the operation was successful return { :ok, identity }, where identity is the identity of the authenticated credential. Otherwise return an error.

    Callback implementation for Gobstopper.Service.Auth.Identity.Credential.authenticate/1.

    Implement the behaviour for changing a credential that is associated with the given identity.

    Callback implementation for Gobstopper.Service.Auth.Identity.Credential.change/2.

    create(identity, arg)

    Implement the behaviour for creating a new credential and associating it with the given identity.

    An identity should only have one credential of type. If the identity is attempting to create a new credential for the same type, an error should be returned.

    If the operation was successful return :ok.

    Callback implementation for Gobstopper.Service.Auth.Identity.Credential.create/2.

    credential?(identity)

    Implement the behaviour for identifying if a credential exists for the given identity.

    If one exists return true, otherwise return false.

    Callback implementation for Gobstopper.Service.Auth.Identity.Credential.credential?/1.

    info(identity)

    Implement the behaviour for retrieving the presentable information for the credential of a given identity.

    If one exists return the state of the credential (:unverified or :verified) and the presentable string. Otherwise return { :none, nil }.

    Verification state is used to infer whether the given credential is guaranteed to be owned by the identity owner.

    Callback implementation for Gobstopper.Service.Auth.Identity.Credential.info/1.

    revoke(identity)

    Implement the behaviour for revoking the credential associated with the given identity.

    If the identity has no credential, then it should return an error.

    If the operation was successful return :ok.

    Callback implementation for Gobstopper.Service.Auth.Identity.Credential.revoke/1.
