    The service application for identities and credentials.

    Configurable Options

    • :server - The name the server should be registered as. This takes a function that accepts a module and returns a valid named server.

    An example configuration:

    config :gobstopper_service,
    server: &(&1)

    Start the service application.

    By default the service will run in :auto setup mode, which will automatically create and migrate the database. To override this behaviour, the mode can be passed in as an argument for the :setup_mode key. The supported setup modes are: :auto and :manual. If using manual mode, the state of the database is left up to the caller.
