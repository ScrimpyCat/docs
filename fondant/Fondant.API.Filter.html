    Link to this function clean(options \\ []) View Source 
    clean(keyword(any())) :: :ok | {:error, String.t()}

    Clean the database.

    Removes all filters and history of the dataset.

    The options field accepts:

    • :server - The server to send the request to.
    • :timeout - The maximum time to wait for a reply.

    If successful returns :ok. Otherwise returns an error.

    Migrate the database.

    Migrates the database to the latest version of the dataset.

    The options field accepts:

    • :server - The server to send the request to.
    • :timeout - The maximum time to wait for a reply.

    If successful returns :ok and the current timestamp. Otherwise returns an error.

    Link to this function migrations(options \\ []) View Source 
    migrations(keyword()) ::
  {:ok, {[integer()], page()}} |
  {:error, String.t()}

    Get a paginated list of migration timestamps.

    The options field accepts:

    • :server - The server to send the request to.
    • :timeout - The maximum time to wait for a reply.
    • :page - The pagination index to retrieve the results of.
    • :limit - The max number of entries to retrieve.

    If successful returns :ok and the list of timestamps for the current page, and index to the next page.

    Returns { :ok, { timestamps, page } } if the operation was successful, where timestamps is the list of integer, and page is the next pagination index. Otherwise returns an error.
