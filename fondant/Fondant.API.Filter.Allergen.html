    Link to this function find(query, options \\ []) View Source 
    find(keyword(String.t()), keyword(any())) ::
  {:ok, {[Fondant.Filter.Allergen.t()], Fondant.API.Filter.page()}} |
  {:error, String.t()}

    Find all allergen filters that match the given query.

    The query field accepts:

    • :any - Search all fields
    • :name - Search all allergen names

    The options field accepts:

    • :server - The server to send the request to.
    • :timeout - The maximum time to wait for a reply.
    • :locale - The localisation Fondant.API.Filter.locale/0 to be applied for the search (see Fondant.Service.Locale).
    • :page - The pagination index to retrieve the results of.
    • :limit - The max number of entries to retrieve.

    Returns { :ok, { allergen_filters, page } } if the operation was successful, where allergen_filters is the list of allergen filters, and page is the next pagination index. Otherwise returns an error.

    get(Fondant.Filter.id(), Fondant.API.Filter.locale(), keyword(any())) ::
  {:ok, Fondant.Filter.Allergen.t()} |
  {:error, String.t()}

    Retrieve a specific allergen filter for the given id.

    If no filter exists with the given id, it will return an error.

    The options field accepts:

    • :server - The server to send the request to.
    • :timeout - The maximum time to wait for a reply.

    Returns { :ok, allergen_filter } if the operation was successful, otherwise returns an error.
